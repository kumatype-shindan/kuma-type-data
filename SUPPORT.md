# Support

Start with the [Kuma Type Shindan guide](https://kumatype-shindan.xyz).

Use GitHub issues for public-safe reports about:

- incorrect question or type data;
- scoring or compatibility calculation mismatches;
- broken public asset references;
- README or documentation corrections.

Do not post private account, payment, analytics, or security-sensitive information in public issues.

For private support, email [Kuma Type Shindan support](mailto:support@kumatype-shindan.xyz).
