# Security

Do not report security issues in public GitHub issues.

Send security-sensitive reports to [Kuma Type Shindan support](mailto:support@kumatype-shindan.xyz). Include affected public URLs, concise reproduction steps, observed impact, and any screenshots or logs with tokens, cookies, account identifiers, and private data removed.

This repository only contains public data, scoring rules, and public reference assets. It does not publish deployment secrets, environment files, payment code, customer data, or private website source.
