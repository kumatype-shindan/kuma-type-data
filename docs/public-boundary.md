# Public Boundary

This repository is intentionally a public data package, not the private website source.

## Public In This Repository

- Quiz questions.
- All 16 Kuma type records.
- Result image references and included public image assets.
- Compatibility lists and scoring helper.
- Selected screenshots for repository documentation.
- Public tests that verify the package shape.

## Not Public Here

- `.env`, `.env.local`, or any runtime secret file.
- Next.js route files, layouts, and UI components.
- Google Analytics implementation details.
- Payment, Stripe, database, auth, webhook, provider, or deployment configuration.
- Vercel project metadata.
- Local debug directories and ad hoc QA captures.
- Private operational notes.
